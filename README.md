# 🌌 Solar System Explorer

A visually appealing HTML and CSS project that displays information about the planets in our Solar System in a modern, responsive table.

## 🚀 Features

- Beautiful space-themed design
- Responsive layout for different screen sizes
- Modern glassmorphism effect
- Gradient table headers
- Hover animations
- Organized planetary data
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📊 Planet Information Included

- Mercury
- Venus
- Earth
- Mars
- Jupiter
- Saturn
- Uranus
- Neptune

For each planet, the table displays:

- Distance from the Sun
- Diameter
- Number of Moons
- Orbital Period
